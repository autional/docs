# Autional Docs

Documentation site for [Autional](https://www.autional.com) — Identity for AI-Generated Applications.

Built with [Astro](https://astro.build). Deployed on [Vercel](https://vercel.com) at **docs.autional.com**.

## Project Docs

- 详细变更说明：`docs/2026-06-30-docs-site-brand-refresh.md`
- UI / 品牌落地规范：`docs/website-ui-brand-guidelines.md`

## Development

```bash
pnpm install
pnpm dev    # http://localhost:4432
pnpm build  # Static output to dist/
```

## Deploy

Push to `main` — Vercel auto-deploys.

## Notes

- 当前本次品牌化改造已通过 `npx astro check`
- 后续继续扩展文档页面时，请优先遵循 `docs/website-ui-brand-guidelines.md`
